# Prints — Live (GitHub Pages)

A modern single-page site that embeds the MJPEG stream at `https://prints.qureshi.io` and includes a “Buy me a coffee” button.

## Customize

Edit `script.js`:

- `COFFEE_URL` → your BuyMeACoffee / Ko-fi / Stripe link
- `REPO_URL` → optional GitHub repo URL

## Deploy to GitHub Pages

1. Create a new GitHub repo (public is simplest for Pages).
2. Upload `index.html`, `styles.css`, `script.js` (and this README).
3. In GitHub: **Settings → Pages**
   - Source: **Deploy from a branch**
   - Branch: `main` / root
4. Your site will appear at: `https://<username>.github.io/<repo>/`

Tip: If you later want a custom domain, add a `CNAME` file.
